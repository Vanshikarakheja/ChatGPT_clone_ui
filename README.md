ChatGPT Clone (Streamlit + SQLite)

A simple ChatGPT-style web app built using Streamlit and SQLite.

🚀 Features
Login / Signup authentication
Multiple chat sessions per user
Chat history saved in SQLite
User profile (name, phone, college)
Basic analytics dashboard
Sidebar for navigation and recent chats
🏗 Tech Stack
Python
Streamlit
SQLite
▶️ Run Project
streamlit run chatgpt_clone_ui.py
🧠 Working
Users create/login accounts
Start new chats or continue old ones
Messages are stored in database
AI responses generated via qa_engine.py
