# ChatGPT UI Clone

A modern ChatGPT-inspired conversational interface built using Streamlit. This project focuses on recreating the user experience and visual design of ChatGPT while introducing additional usability features.

## Features

* Clean and responsive ChatGPT-style interface
* Dark theme design
* Sidebar navigation
* New Chat functionality
* Chat history management using session state
* Interactive prompt suggestion cards
* AI tools section for future feature expansion
* Custom chat timeline with timestamps for tracking conversations
* Reusable and maintainable code structure
* Ready for integration with AI models such as OpenAI, Groq, or other LLM providers

## Unique Addition

The application includes a **Chat Timeline** feature that records conversation prompts along with timestamps, allowing users to easily revisit and track previous interactions. This enhances organization and improves the overall user experience compared to a standard chatbot interface.

## Technologies Used

* Python
* Streamlit
* HTML/CSS (via Streamlit custom styling)

## Future Enhancements

* OpenAI/Groq API integration
* Voice input support
* File upload and document analysis
* Theme customization
* Search across previous conversations

This project demonstrates frontend UI design, user experience considerations, session management, and rapid prototyping using Streamlit.
